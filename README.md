## 如何使用此Hexo博客管理项目
```bash
$ npm install				# 下载hexo博客管理依赖包

$ hexo clean				# 清除之前生成的静态博客

$ hexo generate				# 生成静态文件博客

$ hexo server				# 启动本地博客服务

$ hexo deploy				# 将静态博客发布到gitee

# 刷新gitee pages服务
```
