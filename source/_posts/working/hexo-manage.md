## Hexo博客管理

### 创建一篇文章

``` bash
$ hexo new "文章标题"
```

### 清除上次生成的静态文件

```bash
$ hexo clean
```

### 生成静态文件

``` bash
$ hexo generate
```

### 开启本地web服务

``` bash
$ hexo server
```

### 发布到远程静态文件托管平台

``` bash
$ hexo deploy
```
